# Well-Nest Fitness Website

## Overview
Well-Nest is a beginner-friendly fitness website designed to help users explore simple routines, health tips, and wellness habits.

## Features
- Responsive three-page website
- Home, Routines, and Tips pages
- JavaScript toggle feature for extra wellness tips
- Custom CSS styling using a UNC-inspired color theme

## Tools Used
HTML, CSS, JavaScript

## What I Learned
I practiced semantic HTML, responsive layouts, CSS Grid/Flexbox, and basic JavaScript interactivity.
